## üé® **Design Guidelines**

### **Visual Style**
1. **Consistency**: All badges should follow the same visual style
2. **Readability**: Icons should be clear and recognizable at small sizes (60-80px icon area)
3. **Contrast**: Ensure icons/colors contrast well with badge background
4. **Rounded Corners**: Match app's 3xl rounded corners (or fully circular)

### **Icon Requirements**
- **Size**: 60-80px icon within 120x120px badge
- **Style**: Simple, bold, recognizable
- **Color**: White or light color (`#E8F3E0`) for contrast
- **Thickness**: Medium stroke weight (not too thin, not too thick)

### **Text (Optional)**
- **Font**: Bold, sans-serif
- **Size**: 10-12px
- **Color**: White (`#ffffff`)
- **Position**: Bottom of badge, centered
- **Case**: UPPERCASE

### **Border**
- **Width**: 3px (4px for prestige badges like Champion/Gold)
- **Color**: Based on badge type (see individual descriptions)
- **Style**: Solid

### **Background**
- **Shape**: Circular or rounded square
- **Gradient**: Optional for special badges (Hero, Early Bird, Gold Member)
- **Solid**: For most badges

## üìã **Badge Checklist for Designers**

For each badge, provide:
- [ ] 120x120px PNG file
- [ ] Transparent background
- [ ] Clear, recognizable icon (60-80px)
- [ ] Appropriate color scheme (Seller: Terracotta/Orange, Buyer: Green/Blue)
- [ ] 3px border (4px for prestige badges)
- [ ] Optional: Badge name text at bottom
- [ ] High contrast for visibility
- [ ] Matches app's earth-tone aesthetic

## üí° **Design Tips**

1. **Test at Small Size**: Badges will appear small on profiles, ensure they're readable
2. **Consistent Icon Style**: Use similar line weights and styles across all badges
3. **Color Harmony**: Ensure badges look good next to each other on profile
4. **App Integration**: Badges will appear on:
   - Profile screens (buyer and seller)
   - Rewards/Missions screen (when implemented)
   - Possibly in chat or order details

---

**Note**: All badge designs should align with the app's earth-tone theme while maintaining clear distinction between buyer (green/blue) and seller (terracotta/orange) badges.
